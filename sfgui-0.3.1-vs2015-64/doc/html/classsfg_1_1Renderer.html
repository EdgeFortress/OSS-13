Create and register a new sprite primitive with the renderer.

Sprite primitives are primitives that have a bound texture that was manually loaded prior to creation of the primitive. Subrects can also be specified to use only portions of the passed texture.

Parameters
rectsf::FloatRect describing the rect in which the sprite should be drawn.
textureTexture loaded previously with LoadImage.
subrectSubrectangle for specifying smaller regions of the texture to use. (0,0,0,0) to use whole texture.
rotation_turnsTurns to rotate the texture by in COUNTERCLOCKWISE direction. 1 turn is 90 degrees, -1 turn is -90 degrees etc. 0 to not rotate.
Returns
New sprite primitive.

void sfg::Renderer::AddCharacterSet ( sf::Uint32  low_bound,
sf::Uint32  high_bound 
)

Add a required character set to the character sets that the Renderer will load for new fonts.

This is required if using a script whose glyphs are not smaller than codepoint 0x370 such as Greek, Cyrillic, Hebrew, Arabic etc. By default, only the Latin script and its extensions are loaded.

Parameters
low_boundLower boundary of the character set, i.e. the glyph with the smallest codepoint.
high_boundHigher boundary of the character set, i.e. the glyph with the largest codepoint.
