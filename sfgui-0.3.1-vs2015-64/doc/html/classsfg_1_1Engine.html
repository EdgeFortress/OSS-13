void sfg::Engine::AddCharacterSet ( sf::Uint32  low_bound,
sf::Uint32  high_bound 
)

Add a required character set to the character sets that the Engine will take into consideration for new fonts.

This is required if using a script whose glyphs are not smaller than codepoint 0x370 such as Greek, Cyrillic, Hebrew, Arabic etc. By default, only the Latin script and its extensions are taken into consideration.

Parameters
low_boundLower boundary of the character set, i.e. the glyph with the smallest codepoint.
high_boundHigher boundary of the character set, i.e. the glyph with the largest codepoint.
