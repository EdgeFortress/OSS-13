Detailed Description

Selector.

The selector is used for building paths to widgets using widget names, IDs, classes and states.

bool sfg::Selector::operator== ( const Selector &  other) const

Compare.

Two selectors are equal if they both contain the exact path in the exact order with exact IDs, classes and states set. You can only check two selectors against each other that have been created through Create(). Otherwise this method will always return false.
